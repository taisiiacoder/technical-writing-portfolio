Overview of Swagger UI

Swagger UI is one of the most popular tools for creating interactive documentation. Swagger UI creates an interactive API console for experimenting with real-time requests. In addition, Swagger UI (an actively managed project with an Apache 2.0 license) supports the latest version of the OpenAPI specification (3.x) and integrates with other Swagger tools.

Before we dive into Swagger, we need to clarify some key terms.

Swagger

Refers to the API tools associated with the OpenAPI specification. Some of these tools are Swagger Editor, Swagger UI, Swagger Codegen, SwaggerHub Ð¸ others. All tools are managed by Smartbear. See Swagger Tools for more information. "Swagger" was originally the original name of the OpenAPI specification, but the name was later changed to OpenAPI to reinforce the open, non-licensed nature of the standard. People sometimes refer to both names interchangeably (especially on older web pages), but "OpenAPI" is how the spec should be referred to. For more information on the difference between OpenAPI and Swagger, see «What is the difference between Swagger and OpenAPI?».

OpenAPI

The official name of the OpenAPI specification. The OpenAPI specification provides a set of properties that can be used to describe a REST API. A working, valid document can be used to create interactive documentation, build client SDKs, run unit tests, and more. Details of the specification can be found on GitHub at https://github.com/OAI/OpenAPI-Specification. As part of the Open API initiative with the Linux Foundation, the OpenAPI specification aims to be vendor-independent (many companies are involved in its development).

Swagger Editor

An online editor that checks the OpenAPI documentation against the rules of the OpenAPI specification. The Swagger editor flags errors and provides formatting advice.

Swagger UI

A web framework (on GitHub) that parses a document into an OpenAPI specification and creates an interactive documentation web page. Swagger UI is a tool that turns a spec into a Petstore-like site.
